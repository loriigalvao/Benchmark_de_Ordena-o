📊 Projeto III - Benchmark de Ordenação
Este projeto é uma ferramenta de análise de desempenho que compara diferentes algoritmos de ordenação. Ele permite visualizar na prática qual algoritmo é mais rápido para diferentes volumes de dados.

📝 O que este projeto faz?
Testa Algoritmos: Compara 5 métodos famosos (Bubble, Insertion, Merge, Quick e Heap Sort).

Gera Gráficos: Cria um gráfico de linhas em tempo real para mostrar a curva de desempenho de cada um.

Tabela de Resultados: Exibe o tempo exato em milissegundos (ms) e destaca qual foi o vencedor em cada teste.

Análise Teórica: Inclui uma tabela de consulta sobre a complexidade (Big O) de cada algoritmo.

🛠️ Tecnologias utilizadas
HTML5 e CSS3: Para a estrutura e o visual da página.

JavaScript (ES6): Para a lógica dos algoritmos e cálculos de tempo.

Chart.js: Biblioteca externa utilizada para gerar os gráficos profissionais.

⚙️ Como funciona o teste
O sistema gera listas de números aleatórios com tamanhos de 100 até 10.000 itens e mede quanto tempo o processador leva para organizar tudo.
